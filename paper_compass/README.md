# Paper: Semantic Compasses

## Compile

With ICLR 2026 workshop style (recommended):
1. Place `iclr2026_conference.sty` in this directory (from the ICLR 2026 CFP).
2. Run:
```bash
pdflatex main.tex
bibtex main
pdflatex main.tex
pdflatex main.tex
```

Without the style file, the document compiles in plain `article` class.

## Structure

- `main.tex` — top-level document
- `sections/abstract.tex` — abstract
- `sections/introduction.tex` — intro + contributions
- `sections/background.tex` — OV circuits, SVD, related work
- `sections/methods.tex` — compass plane, causal sweep, null tests
- `sections/results.tex` — gender (3 models) + temporal (Phi-3) + entity (Phi-3)
- `sections/discussion.tex` — limitations, future work
- `sections/appendix.tex` — full logs, SVD scan, prompt lists
- `references.bib` — bibliography
- `figures/` — compass polar/curves/linearity PNGs
