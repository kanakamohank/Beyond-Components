\section*{Impact Statement}

This paper advances mechanistic interpretability by locating
causal low-dimensional features to specific attention heads and
showing that rotations within such a plane produce continuous,
controllable changes in downstream behavior. The same rank-$2$
intervention that \emph{reduces} stereotypical occupation
continuations can, by rotating to the opposite pole, \emph{inject}
them: the primitive is dual-use. We restrict all downstream
evaluations in this work to the Gender Pronoun task and a public
set of occupation prompts, and we do not release steering planes
tuned to harm any demographic group. The persistent
masculine--feminine control asymmetry we document is itself a
finding about how current pretrained models represent gender and
should be accounted for before compass-style interventions are
used as a bias-mitigation tool in deployed systems.
